# Changelog

## 0.1.0 - Developer Preview

- Added Python CLI commands: `doctor`, `probe`, `preview-frame`, and `clean`.
- Added rectangle and polygon mask support.
- Added ProPainter segmented cleanup with ROI padding and scaled repair compositing.
- Added native macOS SwiftUI app for marking masks, configuring paths, running cleanup, and opening outputs.
- Added local unsigned `.app` packaging script.
- Added quickstart, benchmarks, and CI coverage for Python and Swift tests.
